# Video Converter Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add general video converter input formats and output profiles while preserving existing VHS MP4 behavior.

**Architecture:** Keep the shared PowerShell module as the single behavior layer. Extend supported extensions and profile resolution, then update GUI/CLI/release text to present the tool as a broader Video Converter without renaming every compatibility file.

**Tech Stack:** PowerShell, WinForms, FFmpeg, Python pytest.
